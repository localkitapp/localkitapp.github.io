Privacy Policy

Last updated: January 2025

Short version: Your files never leave your browser. No tracking. No uploads. Everything runs locally.

Overview

LocalKit is a browser extension that provides multiple productivity tools including PDF, image, and text utilities. The core principle is local-first: everything runs directly in your browser using native web APIs.

Your files, text, and data are never uploaded or processed on any external server.

Data collection

LocalKit collects almost no data. Specifically:

  • No analytics or tracking
  • No usage data collection
  • No IP address logging
  • No behavioral tracking

The only data stored is:

  • Pro license key (stored locally)
  • User preferences (stored locally)
  • Quick Notes (stored locally)

Files and processing

All files (PDFs, images, text) are processed entirely in your browser using built-in APIs like File API and Canvas.

No file is ever uploaded. Processing happens in memory and results are downloaded directly to your device.

Permissions explained

Permission Purpose Usage
tabs Access open tabs Used only for screenshot feature
activeTab Capture current tab Only when user clicks capture
windows Manage window focus Used for correct capture behavior
storage Store user data Saved locally, never transmitted

Local storage

LocalKit uses browser storage to save:

  • Tool settings
  • Quick Notes
  • Pro license status

All data remains on your device and can be cleared anytime.

Pro license verification

When activating Pro, a one-time request is sent to Gumroad to verify your license key.

No other data is transmitted. After verification, everything works locally.

Third-party libraries

LocalKit includes open-source libraries that run fully locally:

  • PDF.js — PDF rendering
  • pdf-lib — PDF manipulation
  • qrcode.js — QR code generation

These libraries do not make network requests.

This website

This site is hosted on GitHub Pages. GitHub may collect standard logs (IP, browser), but LocalKit does not access or use this data.

No cookies, tracking, or analytics are used.

Changes to this policy

Any updates will be reflected in the “last updated” date.

We will never introduce data collection without clear notice.
