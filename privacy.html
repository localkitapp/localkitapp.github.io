Privacy Policy

Last updated: January 2025

Short version: LocalKit collects no data, sends nothing to any server, and stores nothing outside your own device. Ever.

Data collection

LocalKit collects no personal data of any kind. There are no analytics, no crash reporting, no usage tracking and no telemetry of any kind included in this extension.

We do not know how many people use LocalKit, which tools they use, or anything about how the extension is used. This is a deliberate choice, not an oversight.

Files and content

Any files you open or process with LocalKit — PDFs, images, CSV files, text — are processed entirely within your browser using built-in browser APIs and locally-bundled libraries. No file content is transmitted to any server at any time. Files are read from your local filesystem, processed in memory, and the result is returned to you directly.

LocalKit does not request scripting permission. No scripts are injected into any website you visit. LocalKit does not request broad host permissions (<all_urls>) — network access is restricted exclusively to api.gumroad.com for license verification.

Local storage

Quick Notes uses your browser's localStorage to save notes on your device. This data never leaves your device. Clearing browser data for the extension will delete your notes.

Pro license key is stored in chrome.storage.local after successful verification. It is used only to check your Pro status locally — it is never transmitted except during the single Gumroad API call when you first activate.

Onboarding state (whether you have completed the first-run tour, total tool uses for review nudge timing) is stored in chrome.storage.local. It never leaves your device.

Pro license verification

When you activate a Pro license key, LocalKit makes a single HTTPS request to api.gumroad.com to verify the key. This request includes your license key and the LocalKit product ID. No other information is sent — no browser fingerprint, no IP logging on our end, no personal data.

After verification the result is cached locally and re-verified silently every 30 days. If Gumroad is unreachable during re-verification your Pro status remains active — we never revoke access due to a network error.

Third-party libraries

LocalKit bundles the following open-source libraries. All run locally and make zero network requests:

  • pdf-lib.min.js — PDF creation, merging, splitting and manipulation
  • pdf.min.js + pdf.worker.min.js (PDF.js by Mozilla) — PDF rendering and page thumbnail preview
  • qrcode.min.js (qrcodejs) — QR code generation

This website

This website (localkitapp.github.io) is hosted on GitHub Pages. GitHub may collect standard server access logs (IP address, user agent, referring URL) as part of their hosting service. We do not control this and do not access these logs. See GitHub's privacy statement for details.

This website uses no analytics, no cookies and no tracking scripts.
