Convert images into a PDF document

Convert JPEG, PNG and WebP images into a PDF. Each image becomes one page. Control page size, orientation, image fit and margins. Powered by pdf-lib running entirely in your browser.

LocalKit · Image to PDF
Image to PDF · 100% local · nothing uploaded

How to use

1
Add images
Drop JPEG, PNG or WebP images. Drag cards to reorder — the order sets the page order.
2
Set page size
A4, Letter, or Fit image (page matches the image dimensions exactly).
3
Choose image fit
Contain (letterbox), Fill (crop to fill), or Stretch (distort to fill).
4
Convert and download
The PDF downloads with a filename based on your document title setting.

Settings

SettingOptions
Page sizeA4 (595×842pt), Letter (612×792pt), Fit image
OrientationPortrait, Landscape
Image fitContain, Fill, Stretch
Margin0–80px around the image on each page
Quality10–100% — affects JPEG embedding compression
BackgroundWhite, Black, Transparent
Document titleWritten to PDF metadata, used in filename
WebP images are automatically converted to PNG before embedding, since PDF does not natively support WebP.
