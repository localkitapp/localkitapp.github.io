WiFi QR codes

WiFi QR codes follow the standard WIFI:T:WPA;S:ssid;P:password;H:false;; format. Android and iOS can scan these to connect automatically without typing a password.

For WPA2/WPA3 networks select WPA. For open networks with no password select None.

Error correction

Higher error correction means the QR code can still be scanned even if part of it is damaged or obscured — but it also makes the code denser. Use H if you plan to print and the code might get worn.

Download formats
