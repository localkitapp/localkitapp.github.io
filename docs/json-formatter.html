JSON Formatter

Format, validate and explore JSON

Five tabs in one — Format, Tree, Query, Convert and Diff. Paste any JSON, pretty-print with syntax highlighting, navigate the structure in a collapsible tree, run JSONPath queries, convert to YAML or CSV, and diff two JSON blobs side by side. Runs entirely in your browser.

Format tab

Paste JSON into the input panel on the left. Live line numbers sync with scroll. As you type the status bar reports any parse error with the browser's exact message.

Actions

ButtonShortcutWhat it does
FormatCtrl/Cmd + EnterPretty-print with syntax highlighting and the selected indent size.
Minify—Collapse to a single line, removing all whitespace.
Sort keys—Recursively sort all object keys A→Z.
Validate—Report whether the JSON is valid and show root type and key count.

Indent

Toggle between 2 spaces, 4 spaces or Tab indent. The setting applies on the next Format action.

Output panel

The formatted output is syntax-highlighted: keys in purple, strings in green, numbers in amber, booleans in orange, null in red. Use Copy to copy output, Save to download as .json, or Tree → to jump straight to the Tree tab with the current JSON pre-loaded.

Input controls

Use Paste to paste from clipboard, Open file to load a .json or .txt file, or drop a file anywhere on the page. Clear resets both panels.

Tree tab

An interactive collapsible tree of the full JSON structure. Every node shows its type badge — {obj} for objects, [arr N] for arrays with item count. Hover any row to reveal two inline buttons:

  • 📋 Copy value — copies the node's value (or the full subtree as JSON) to clipboard
  • 🔗 Copy path — copies the JSONPath string to reach this node

The current node path is shown in the toolbar as you hover. The search box highlights all matching tree rows — parent nodes expand automatically so no result is hidden.

Query tab (JSONPath)

Type a JSONPath expression and press Run or Enter. A result count badge shows how many nodes matched.

ExpressionMeaning
$Root element
.keyChild named key
..keyRecursive descent — finds key anywhere in the tree
.*All children (wildcard)
[n]Array element at index n
[-1]Last array element
[0:3]Slice — elements 0, 1, 2
[?(@.age)]Filter — items that have an age property
[?(@.age>18)]Filter with condition

Click any example button in the tab to load it into the query field instantly.

Diff tab

Paste two JSON blobs into the JSON A and JSON B panels and click Diff. LocalKit performs a deep recursive diff and highlights every change:

ColourSymbolMeaning
Green+Key added in B
Red−Key removed — only in A
Amber~Value changed between A and B
GreyUnchanged

A summary strip at the top shows total added, removed and changed counts. The diff output can be copied to clipboard.
