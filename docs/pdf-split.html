Page selection

Thumbnails of every page render in a grid. Click to select or deselect individual pages. Use the toolbar shortcuts for quick selection:

  • Select all — selects every page
  • Clear — deselects everything
  • Odd — selects pages 1, 3, 5, 7...
  • Even — selects pages 2, 4, 6, 8...

Range syntax

In range mode, enter page numbers separated by commas. Use a hyphen for a range:

  • 1-5 — pages 1 through 5
  • 1, 3, 5 — pages 1, 3 and 5
  • 1-3, 8, 10-12 — pages 1–3, page 8, pages 10–12
Password-protected PDFs cannot be split. The tool will detect encryption and show an error.
