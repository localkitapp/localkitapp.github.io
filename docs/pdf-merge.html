PDF Merge

Combine PDFs locally

Merge multiple PDF files into one. Drag to reorder pages, preview the first page of each file before committing. All processing uses pdf-lib running in your browser — your documents never leave your device.

How to use

1
Add PDFs
Drop PDFs onto the drop zone or click to browse. Free tier allows up to 3 files. Pro removes this limit.
2
Reorder by dragging
Drag the handle on the left of each card to change the order. The order you see is the order in the merged file.
3
Preview pages
Click any file card to preview its first page on the right. Encrypted files cannot be previewed but can still be merged.
4
Click Merge PDFs
The merged file downloads automatically. Filename includes a timestamp.

Notes

  • Password-protected PDFs are detected automatically — a warning shows on the card
  • Encrypted files may be skipped during merge depending on the encryption method
  • Duplicate files (same name and size) are rejected automatically
  • Page count is read by PDF.js — loading appears briefly on each card
PDF merging uses pdf-lib, a pure JavaScript library bundled with the extension. No server, no third party.
