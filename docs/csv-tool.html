Opening a file

Drop a .csv file onto the drop zone or click Open CSV in the toolbar. LocalKit auto-detects the delimiter — comma, semicolon or tab — and parses quoted fields correctly including embedded newlines and escaped quotes.

You can drop a new file at any time while the table is open. It replaces the current file immediately and resets all filters and sort state.

Sorting

Click any column header to sort ascending. Click again to sort descending. Numeric columns sort by value rather than alphabetically. The active sort column shows a direction arrow.

Filtering and search

Use the All columns dropdown and filter input to filter rows by a specific column or across all columns at once. Filter applies live as you type.

The search box highlights matching cells across all visible rows and shows a match count. Press ✕ to clear.

Inline cell editing

Double-click any cell to edit it. The cell switches to a text input:

  • Enter — commit the edit
  • Escape — cancel and revert to the original value
  • Tab — commit and move to the next cell

The status bar shows unsaved* when there are uncommitted edits. Use Export → CSV to save changes back to a file.

Find & Replace

Click Replace in the toolbar or press Ctrl/Cmd + H to open the Find & Replace panel. It slides up from the bottom of the screen above the status bar.

  • Find all — counts matching cells and highlights matching rows in the table
  • Replace all — replaces every match across all rows and columns in one operation
  • Case sensitive checkbox — toggle exact-case matching

Duplicate detection

Click Dupes in the toolbar to highlight all rows that are exact duplicates (every column must match). Duplicate rows are tinted. Click Dupes again to clear. The status bar shows the count of duplicate rows found.

Column statistics

Click Stats to open the statistics panel. For each column:

  • Total rows, non-empty count, empty count, unique value count
  • Numeric columns — min, max, average, sum
  • All columns — top values bar chart (up to 7 most frequent values)

Undo

Press Ctrl/Cmd + Z to undo the last action. LocalKit keeps up to 50 undo states per session. Undo covers cell edits, row insertions, deletions, duplications and Replace all.

Export

Click Export to open the export menu. All exports respect the current filter — only visible rows are exported when a filter is active.

Limitations

  • Maximum recommended file size: 50 MB. Larger files may render slowly.
  • Column widths are per-session only and reset when a new file is opened.
  • All edits are in-memory. Use Export → CSV to save changes to disk.
