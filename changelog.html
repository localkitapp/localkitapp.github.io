v1.0.0 Initial release 2 April 2026
Free tools — 6 tools shipped
NEW
Screenshot — capture any browser tab as PNG or JPEG. Copy to clipboard or download. Uses Chrome captureVisibleTab API. Nothing uploaded.
NEW
PDF Merge — combine up to 3 PDFs into one using pdf-lib running entirely in the browser. Drag-to-reorder, first-page preview, 50 MB per file limit.
NEW
Image Compressor — compress JPEG, PNG and WebP using Canvas API. Interactive before/after drag slider. Compression quality slider with live size preview.
NEW
Quick Notes — persistent scratchpad stored in localStorage. Auto-save every 600 ms, pin notes, search, word count per note. Survives browser restarts.
NEW
Password Generator — cryptographically secure passwords via crypto.getRandomValues. Strength meter by entropy bits. 4–64 character length. 4 presets.
NEW
Word Counter — 11 live text statistics including words, characters, sentences, reading time, speaking time, unique words. Live platform limits for Twitter, LinkedIn, Instagram.
Pro tools — 7 tools shipped ($9 one-time)
NEW
QR Generator — URL, plain text, WiFi credentials, email with prefilled fields. Custom foreground/background colours. PNG and SVG export. L/M/Q/H error correction.
NEW
PDF Split — extract specific pages or split every page into individual files. Page range selection. All processing via pdf-lib — no upload.
NEW
Screenshot Annotate — full-screen canvas editor with 8 annotation tools (pen, arrow, rectangle, ellipse, text, blur/redact, highlight, select). 50-state undo/redo. Full keyboard shortcuts.
NEW
Image to PDF — convert multiple JPEG, PNG or WebP images into a single PDF. A4, Letter, or fit-to-image page sizing. Drag-to-reorder images before converting.
NEW
Image Tools — batch resize (by width, height, exact, or percent) and bulk format conversion (JPEG/PNG/WebP). Aspect ratio lock. Processes multiple images at once.
NEW
Text Cleaner — 20+ cleaning operations: fix smart quotes, em dashes, zero-width chars, non-ASCII, sort/dedupe/reverse lines, case conversion. 4 one-click presets for PDF, Word, email, and plain text.
NEW
PDF Merge Unlimited — Pro version of PDF Merge with no file count limit. Same drag-to-reorder and preview interface.
Foundation
SECURITY
All tools verified to make zero network requests for user data — confirmed via Chrome DevTools Network tab. Third-party libraries (pdf-lib, qrcodejs, PDF.js) run entirely locally.
NEW
Pro license system — Gumroad-based one-time purchase. License key stored in chrome.storage.local. Re-verified silently every 30 days. Pro access retained if Gumroad is unreachable.
NEW
Manifest V3 — built on Chrome's latest extension standard with minimal permissions. Only storage, tabs, activeTab, scripting, windows.
